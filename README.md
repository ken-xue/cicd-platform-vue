## 项目简介

> devops-platform-vue 是一款企业级开发、效能、运维平台。打通项目立项、研发、测试、发布、运维等流程形成闭环。前后端分离开发[后端地址](https://github.com/ken-xue/devops-platform)


## 目录结构

```
ddd-framework-vue
    └──src        源码
       ├── api
       ├── assets
       ├── components
       ├── directive
       ├── filters
       ├── icons
       ├── layout
       ├── main.js
       ├── permission.js
       ├── router
       ├── settings.js
       ├── store
       ├── styles
       ├── utils
       ├── vendor
       └── views
```

## 快速开始

```bash
git clone https://gitee.com/ken-xue/devops-platform-vue.git
cd devops-platform-vue
npm install
npm run dev
```
